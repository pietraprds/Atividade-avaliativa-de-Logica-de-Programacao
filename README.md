

# Atividade Avaliativa 1 - Lógica de Programação com Python 🐍

Este repositório contém a resolução de uma atividade prática focada em conceitos fundamentais de Python, como **estruturas de repetição (loops)**, **condicionais (if/elif/else)** e **manipulação de listas**.

## 📋 Descrição das Questões

O arquivo `Atividade_Avaliativa_1.ipynb` está dividido em quatro problemas práticos de automação e análise de dados simples:

### 1. Classificação de Temperaturas
Um script que percorre uma lista de temperaturas e as classifica em três categorias:
* **Frio:** Abaixo de 20°C.
* **Agradável:** Entre 20°C e 30°C.
* **Quente:** Acima de 30°C.
O código também realiza a contagem de ocorrências em cada categoria.

### 2. Sistema de Avaliação de Alunos
Simula o processamento de notas de uma turma. O sistema define:
* **Reprovado:** Nota < 5.
* **Recuperação:** Nota entre 5 e 7.
* **Aprovado:** Nota > 7.
Ao final, o script exibe a quantidade total e a porcentagem de alunos aprovados.

### 3. Monitoramento de Consumo de Energia
Analisa o consumo energético semanal, separando os valores em listas de consumo **Baixo**, **Moderado** ou **Alto**. Além da classificação, o script calcula:
* Soma total do consumo da semana.
* Média diária de consumo.


### 4. Simulação de Carrinho de Compras
Um sistema de descontos progressivos para uma lista de produtos. A lógica aplicada para os preços é:
* **5% de desconto** para itens abaixo de 50.
* **10% de desconto** para itens entre 50 e 150.
* **15% de desconto** para itens acima de 150.
O programa gera uma nova lista com os preços atualizados e calcula o valor total da compra.
